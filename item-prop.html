Flex-grow

This property estamblishes the ability for a item to grow if needed. A unitless number serves as a proportion of the spacing of the item. It represents the horizontal space being taken up by the items to cover the given space.

.container {
flex-grow: 2; /* number */
}

Center
  • Flex-start: Item is placed near the cross start.
  • Flex-end: Item is placed near the cross end.
  • Center: Evenly centers the item between the cross start and end.
  • Stretch: Streches the whole item to cover from the cross start to the end.
  • Baseline: aligns the item based off the the baseline of the first item.
  • float, clear and vertical-align do not have an effect on a flex item.
