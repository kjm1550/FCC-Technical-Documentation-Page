Display

This property defines the flex container, parent. It also defines the flex item, children, to all of its direct children. This property is necessary to use flexbox.

.container {
display: flex; /* or inline-flex */
}

Flex-direction

This property estamblishes the direction the flex items are placed and of the main axis.

Flex-wrap

The default for flex is to put every flex item in one row. This can be changed with this property.

.container{
flex-wrap: nowrap; /* wrap or wrap-reverse */
}

Wrap
  • Nowrap: Default. One row for all the flex items.
  • Wrap: Flex items will wrap to multiple rows if needed.
  • Wrap-Reverse: Same as wrap, but bottom to top.
Flex-flow

This property joins flex-direction and flex-wrap into one property. It defines the main and cross axis. The two properties are seperated by a space.

.container{
flex-flow: /* flex-direction flex-wrap */
}
Justify-content

This property sets the alignment along the main axis. It helps distribute space when items do not fill the maximum size of the row.

.container {
justify-content: /* flex-start, flex-end, center, space-between, space-around, and space-evenly */
}

Space

Between
  • Flex-start: Default. Items are placed near the main start.
  • Flex-end: Items are placed near the main end.
  • Center: Evenly centers the items between the main start and end.
  • Space-between: Items are evenly distributed accross the main axis. The first item is on the main start, last item on the main end.
  • Space-around: Space is disributed around all the items, the main start, and main end.
  • Space-evenly: Space between any items, main start, and main end are evenly distributed.
Align-content

This property aligns rows of items between the cross start and cross end. Like what justify-content does to items on the main axis, this does to rows on the cross axis. Multiple rows of items.

.container {
align-content: /* flex-start, flex-end, center, space-between, space-around, or stretch */
}

Flex-end
  • Flex-start: Rows are placed near the cross start.
  • Flex-end: Rows are placed near the cross end.
  • Center: Evenly centers the rows between the cross start and end.
  • Space-between: Rows are evenly distributed accross the cross axis. The first item is on the cross start, last item on the cross end.
  • Space-around: Space is disributed around all the rows, the cross start, and cross end.
  • Stretch: Default. Space between any rows, cross start, and cross end are evenly distributed.
Align-items

This property defines how items are layed out along the cross axis. Similar to align-content but just one row of items.

.container {
align-items: /* flex-start, flex-end, center, baseline, or stretch */
}
